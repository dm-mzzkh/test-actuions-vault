All the everyday Markdown you'd expect, plus a few Obsidian extras.

## Inline styles

- **Bold**, *italic*, ***bold italic***
- ~~Strikethrough~~
- ==Highlighted text== (Obsidian's `==` syntax)
- `inline code`
- A footnote reference[^1] sits inline and links to the bottom.

## Lists

Unordered:

- First
  - Nested
    - Deeper

Ordered:

1. Step one
2. Step two
3. Step three

## Task lists

- [x] Set up the vault
- [x] Configure `baseUrl`
- [ ] Write more notes
- [ ] Share the link
