This folder is a **tour** of the syntax Quartz supports. This very page is a
*folder note* — because it's named `index.md` inside the `features/` folder,
Quartz uses it as the landing page for the folder instead of an auto-generated
list.

## Pages in this tour

- [[links]] — connect notes with `[[wikilinks]]` and see [[links#Backlinks|backlinks]]
- [[callouts]] — colorful `> [!note]` admonitions
- [[formatting]] — bold, highlights, tables, task lists, footnotes
- [[code-and-math]] — fenced code blocks and `$LaTeX$`
- [[diagrams]] — Mermaid flowcharts and more
- [[media]] — images and note embeds
- [[tags-and-metadata]] — frontmatter and `#tags`

1. Click any link above (notice the **popover preview** on hover).
